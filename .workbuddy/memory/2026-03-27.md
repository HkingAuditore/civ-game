## 经济崩溃模型分析报告
- 生成交互式 HTML 仪表盘报告：ai_reports/economic_collapse_report.html
- 数据源：388个崩溃会话（214玩家）、306个正常对照组、75万+ design_events
- 核心发现：
  - 饥荒是最常见崩溃态（352次），远超恶性通胀（99次）和破产（35次）
  - 通胀型崩溃前 CPI 稳步上升，破产型崩溃前 CPI 反而暴跌——两种不同路径
  - 国库储备不足是最显著的崩溃先兆（崩溃组中位数 42K vs 对照组 374K，9倍差距）
  - CPI 单次增速 >20% 是最有效的预警信号
  - GDP-人口相关系数从正常组 0.983 降到崩溃组 0.946，脱钩是危险信号
  - 失业者(293次)和工匠(200次)是最易叛乱阶层
