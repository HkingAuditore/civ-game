## [2.0.9] - 2026-03-17

### Fixed
- **[2026-03-17] 停战与价格模型修复**
  - 修复接受敌方停战提议时未统一复用战争结束逻辑的问题，避免外交关系与停战状态处理不一致
  - 修复营业税为负值（补贴）时成本价可能小于 0 的问题，成本价现在保底为 0
  - 清理雇员工资冷启动推导处的注释串行问题，降低后续维护误判风险

### Changed
- **[2026-03-17] AI 时代推进与难度经济波动平衡**
  - 上调 AI 各时代科技与文化升级需求系数，显著放缓 AI 冲时代速度
  - 时代升级冷却由 `200` 提升至 `600`，科技文化累积速率缩放由 `1.0` 下调至 `0.4`
  - 重做难度库存目标倍率：和平/简单模式提高库存缓冲，困难及以上模式降低库存缓冲，强化难度差异
  - 建筑利润计算统一通过税务模块读取营业税倍率，确保营业税倍率被正确限制在合法范围
