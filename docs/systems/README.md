## 核心机制速查

### 满意度公式
```
目标满意度 = 生活水平基础(70-95) + 税负修正 + 短缺惩罚 + 财富趋势
实际满意度 += (目标 - 当前) × 0.02  每日渐变
```

### 产量公式
```
产量 = 基础产出 × (1 + Σ加成) × 填充率 × 原料系数 × 利润边际
```

### 生活水平评分
```
总分(0-100) = 收入充裕度(0-50) + 需求满足率(0-30) + 财务安全度(0-20)
```
