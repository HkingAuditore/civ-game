# 🎮 文明崛起 - 新 UI 快速体验指南

## 🚀 启动游戏

```bash
# 开发模式（支持热更新）
npm run dev

# 构建生产版本
npm run build
npm run preview
```

## 📱 移动端体验

### 方式 1：浏览器开发者工具
1. 打开游戏（http://localhost:5173）
2. 按 `F12` 打开开发者工具
3. 点击 "Toggle device toolbar"（或按 `Ctrl+Shift+M`）
4. 选择设备：iPhone 12 Pro / Pixel 5 等

### 方式 2：真实手机测试
1. 确保手机和电脑在同一 WiFi 网络
2. 启动开发服务器：`npm run dev -- --host`
3. 在手机浏览器访问：`http://[你的电脑IP]:5173`
4. 例如：`http://192.168.1.100:5173`

## 🎯 新功能体验清单

### ✅ 移动端特性

1. **顶部状态栏**
   - 点击时间图标 → 展开资源列表
   - 点击银币 → 查看资源详情
   - 点击收支趋势 → 查看税收详情

2. **底部导航栏**
   - 5 个主要标签页图标
   - 选中状态会发光
   - 方便单手操作

3. **帝国场景**
   - 显示在主内容区顶部
   - SVG 动画场景

4. **游戏控制**
   - 右上角浮动按钮
   - 速度控制、存档、帮助

### ✅ 桌面端特性

1. **三列布局**
   - 左侧：资源、阶层、手动采集
   - 中间：标签页内容
   - 右侧：场景、日志、提示

2. **顶部导航**
   - 状态栏 + 游戏控制
   - 标签页横向导航

3. **玻璃拟态效果**
   - 半透明背景 + 模糊效果
   - 高光边框和发光动画

## 🎨 设计亮点

### 1. 玻璃拟态 (Glassmorphism)
- 半透明背景：`bg-gray-900/80`
- 模糊效果：`backdrop-blur-md`
- 细边框：`border border-white/10`

### 2. 发光效果
- 按钮悬停：`hover:shadow-glow-sm`
- 选中状态：`shadow-glow-md`
- 动态主题色（根据时代变化）

### 3. 平滑动画
- 所有交互都有过渡效果
- 弹窗滑入动画
- 数字变化动画

## 🐛 已知问题

1. **首次加载**
   - 可能需要几秒钟加载资源
   - 建议使用生产构建测试性能

2. **移动端横屏**
   - 目前主要优化竖屏体验
   - 横屏布局会自动切换到桌面端布局
