# 2026-03-13 工作日志

## 产业链 Tab 重构 (ResourceDetailModal.jsx)

- **删除旧组件**: `buildChainPaths`（DFS算法）、`ChainFlowDiagram`（横向节点流）、`SimpleBuildingChain`（fallback上下游）、旧版 `DynamicChainView`（共约350行）
- **新增组件**: `IndustryChainView`（主容器）、`ChainCard`（单条产业链卡片）、`StageRow`（阶段行）、`ResourceChip`（资源标签）、`BuildingChip`（建筑标签）
- **核心改进**:
  - 抛弃 DFS 全路径遍历，改为以资源为锚点只提取直接上下游一层关系（O(n)复杂度）
  - 垂直卡片布局替代横向滚动，天然适配手机
  - 每张卡片显示：产业链名称 + 资源角色标签（原材料/中间品/产出品/被消费）
  - 三段式结构：上游(生产来源) → [当前资源高亮] → 下游(消耗去向)
  - epochRange 过滤：未解锁时代的 stage 半透明显示
  - 建筑信息收敛为小标签
- **保持兼容**: `DynamicChainView` 组件名和 props 接口不变，调用入口无需修改
- **辅助工具函数**: `getResourceRole`(角色分析)、`getDirectRelations`(上下游提取)、`ROLE_CONFIG`(角色样式配置)
